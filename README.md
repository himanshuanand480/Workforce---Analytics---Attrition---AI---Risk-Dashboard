# Workforce Analytics Dashboard: Attrition vs AI Risk

## 📌 Project Overview
This project analyzes employee attrition based on AI risk, salary, job satisfaction, and experience levels.

## 🎯 Objective
To identify how AI automation and employability gap impact employee attrition.

## 🛠️ Tools & Technologies
- Excel (Data Cleaning)
- MySQL (Data Storage & Queries)
- Python (EDA & Visualization)
- Power BI (Dashboard)

## 🔗 Data Pipeline
Excel → MySQL → Power BI  
Python used for analysis (Kaggle Notebook)

## 📊 Key Insights
- Employees with low salary show higher attrition
- High AI-risk roles have more attrition
- Mid-level employees (3–6 years) are most affected
- Data & AI roles show lower automation risk
